The Window Object

The window object is standard across all browsers

The Browser Object Model is not standard and has info about the screen and browser

There is no standard, but many browsers support common methods/properties

Global variables are acutally part of a global object in the Window object

Window Object functions:

.navigator.userAgent - get the browser

.location.href - current URL

.location.hostname - name of the domain

.history.go(1) - goes forward 1 page

.open(url, title, attributes) - opens a new window

.close() - close a window

.moveTo(x,y) - position window on screen

.screen.height - pixel height of screen

Cookies are saved locally on the users computer

Max storage of 4KB, 20 cookies per domain

Can be used to store information across visits

document.cookie = 'key=value'; - Store a cookie

Assinging new cookies will only append

Use the same key to assign new values

cookies are stored in a string like this: "key=value; key=value; key=value"

document.cookie.split("; ") to separate cookies

for (crumb of cookies){const [key,value] = crumb.split("=");} - Assigns pairs

Expiration dates can be set using a key-value pair "expires=time" (time is seconds)

cookies can be assigned to a path or domain as well
