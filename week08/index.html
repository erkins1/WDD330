Transforms and Transitions

CSS3 transform property allows elements to translate, rotate, scale and skew

Translation: moves elements left, right, up and down

Scaling: scales elements horizontally then vertically

Rotation: rotates element around point using degrees

Skew: skew the element along X and Y axis

The origin of the transformation can be changed

Transitions change over time to create animations

Creating a transition requires elements to have:

1. An original/default state

2. A final state

3. Transition function using transition properties

transition-property: defines element properties that should change

transition-duration: how long the transition should take

transition-timing-function: control the pace of the transition

transition-delay: time before the transition begins

shorthand: "transition: [property] [duration] [timing] [delay]"

Transitions are only a start and end, animations define intermediate steps

Animations use keyframes which are a points inbetween the start and end states

Canvas, SVG and Drag and Drop

Canvas allows us to draw with javascript

use the canvas html tags to define the draw area

Canvas has many features that can be used including saving, pictures, and video

SVG stands for Scalable Vector Graphics

SVG uses the DOM and XML tags

Other websites exist to help make these SVG objects easy to create

Canvas is easy to draw pictures, but cannot be changed once created

SVG lets you change aspects of pictures drawn be it

Drag and Drop API allows us to identify elements that can be dragged/dropped

This is not suppoered by Android or iOS

Elements have to be enables to be dragged or receive dropped elements
