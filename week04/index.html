Forms

Forms help to control input fields

Usually php is used to send information back. JavaScript is used to do some front end processing first

Forms can be accessed using document.forms[0] or document.forms['formName']

A form object contains its elements

Form Events:

focus - when cursor in inside the form element

blur - when cursor leaves the form element

change - form elements value is changed

submit - when the form is submitted

From Controls:

input

select

Object Oriented Programming

Encapsulation - Logic and complex functions are hidden

Polymorphism - Objects can share the same methods, but can override them

Inheritance - Objects can include a prexisting object

static methods are called by the class, not by an instance of the class

Prototypes are consistent across instances of an object

public methods and properties are queried directly

private methods and properties are not accessable from outside the class

prototypes can be used to implement ineritance

enumberable allows properties to show up in a loop

Modular JavaScript

modules are always in strict mode

global variables defined in the module exist only in the module

default export refers to a vaiable, method or class in a module that can be imported without specifying

Node.js requires the use of module.exports to be accessable
