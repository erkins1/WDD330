Basics

Global variables are declared outside of a function or block. However using globals is generally bad practice.

Template Literals are ways to insert JavaScript inside a string. The '${ }' is used to differentiate the JavaScript code.

The backslash can be used to allow special characters in a string.

Arrays, Sets and Maps

Arrays can be declared: const myArray = []; or const myArray = new Array();

Multidimensional Arrays are declared: const myArray = [[],[]];

Special Array Methods:

array.pop() Removes the last item from array

array.shift() Removes the first item from array

array.push('var') Appends a new value to the end of array

array.unshift('var') Appends a new value to the beginning of array

array.concat([new array]) Combines two arrays

array.join('&') Changes array into string separated by '&' (Default is ',')

array.slice(2,4) Copies index 2 - 4 of the array

array.splice(2,1,'var') Replaces index 2 with new element

array.splice(2,0,'var') Inserts new element into index 2 without removing elements

array.splice(2,1) Removes index 2

array.reverse() Reverse the order of the array

array.sort() Sorts the array alphabetically

array.indexOf('var') Finds the index of the parameter

array.includes('var') Returns if array has a parameter

array.includes('var', 3) Returns if array has parameter after index 3

A Set is essentially an array with no duplicates

Maps are lists of key value pairs. Essentially a dictionary.

Maps are declared: const myMap = new Map();

Entries are added: myMap.set(1, 'I');

Use keys to get the associated value: myMap.get(4);

Other Map methods are in JavaScript: Novice to Ninja p.g. 112-115

Functions

There are a few ways to define a function:

Function Literals function hello(){ some code; }

Function Expressions const goodbye = function(){ some code; }

const goodbye = function bye(){ some code; };

Function Constructor const hi = new Function('some code;');

Arrow Functions const square = x => x*x;

const add = (x,y) => x+y;

const hello = () => { const helloWorld = 'Hello World'; alert(helloWorld); }

Be aware of function hoisting when setting a function to a variable.

You can pass functions into a method by including a callback parameter that will accept a function.
