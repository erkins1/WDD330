Objects

Objects are created: const objectName = {};

Properties can be accessed with dot or bracket notation

Can also have computed properties p. 171

If a property doesn't exist, it will return undefined

Objects can contain other objects (nested)

get and set methods help to access private variables

The Math object is a good example of an object

Document Object Model

The DOM is an object that represents the associated html doc

Like other objects, the DOM has attributes and methods allowing us to change the object

There are lots of methods to learn (p. 216)

Events

Event listeners allow functions to run when the user preforms an action

This is better than an infinate loop that looks for an action to be preformed

Events can be added like this: document.body.addEventListener("click", doSomething);

HTML events can be placed in the html tags with an onclick attribute: <p onclick="console.log('You Clicked Me!')">

Use event.target to know what triggered the event

Use event.pageX, event.pageY, event.screenX, event.screenY to know corrdinates

Types of events:

click, mousedown, mouseup, dblclick, mousemove, keydown, keyup, keypress, touches, touchend,
